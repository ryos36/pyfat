オリジナルの pyfat から大幅に変えています。
いくつかのファイルからそれを含む fat32 + vfat の FAT を作ることが所期の目的で、
フレキシブルではありませんが、それを達成してしまったのでそこから更新してません。
そのため、暫定的なコードが散らかっています。
だれかフレキシブルにしてください。

make.sh で空の dosfs のイメージを作ります。
その後、mkfat.py で 

- "sdcard/boot.bin"
- "sdcard/uramdisk.image.gz"
- "sdcard/devicetree.dtb"
- "sdcard/uImage"

をその dosfs に書き込みます。
これにより Zynq 用の sdcard イメージを自動的に作ることができます。

mkfat.py の中ではちゃんとセクタを見てません。このファイルだけなら１セクタ内に
収まるはずというかなり楽観的なつくりになっています。厳密にやるなら
セクタ長のチェックをする必要があるでしょう。
